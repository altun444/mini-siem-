 # Mini SIEM (Security Information and Event Management)

A lightweight, high-performance Mini SIEM system designed for real-time log ingestion, signature-based threat detection, and automated alerting. This project serves as a practical blueprint for understanding defensive cyber security engineering and event correlation workflows.

## Features
- **Centralized Log Ingestion**: Receives and processes system events from remote endpoint agents.
- **Signature Detection Engine**: Evaluates incoming logs against dynamic security rules to catch indicators of compromise (IoCs).
- **Extensible Architecture**: Built with modular components, making it easy to integrate secondary databases or third-party webhooks.
